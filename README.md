# Tutorial Machine Learning amb Scikit-Learn

Aquest és un projecte d'un estudiant de 1r de Batxillerat de l'institut Vinyes Velles on s'anirà veient el progrés que té aquest repositori per acabar tenint un tutorial que té com a objectiu aconseguir mostrar a un novell el mes bàsic de l'aprenentatge automàtic.

El tutorial en markdown es pot trobar a la carpeta tutorial.

**Aquesta és la taula de continguts:**

1. Tipus d'algoritmes ( Supervisat o no supervisat)
2. Visualització de Dades
3. Normalització de Dades
4. Algorismes Supervisats
   - Regressió Lineal
   - Classificació
5. Algorismes No Supervisats
   - Clustering (K-Means)
   - Reducció de Dimensionalitat (PCA)
6. Reconeixement d'Imatges (SVM)


Al repositori es pot trobar també una carpeta nomenada code, en la que hi ha dintre els diferents codis utilitzats per poder fer aquest projecte, es facilitat aquests codis ja que aixi es pot entre millor el funcionament sense tant de text. Principalment s'ha utilitzat el breast_cancer dataset, pero en el SVM s'ha utilitzat el de digits.

També s'ha de dir que aquest és un projecte fet per un principiant per a altres principiats, pot contenir errors, tant de codi, com d'explicació. Compte amb això.
